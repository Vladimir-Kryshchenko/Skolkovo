# База Сколково

RAG-база знаний по документам Фонда «Сколково» с **открытым MCP-доступом** для агентов и приложений.

## Возможности

- 📥 **Сбор документов** с https://dochub.sk.ru/foundation/documents/
- 🔄 **Регулярный парсинг** сайта на новости, события и изменения
- 🧠 **RAG-база** — векторный поиск и извлечение по документам
- 🔌 **Открытый MCP-сервер** — подключение любых агентов и приложений
- 🛠 **Админка** — классификация, валидация, контроль статуса (действует / устарел)

## Структура репозитория

| Папка | Назначение |
| :--- | :--- |
| `Правила_проекта/` | Правила ведения проекта |
| `Управление_проектом/` | Бэклог, планы и отчёты по спринтам |
| `Аналитика/` | Архитектура, ТЗ, спецификации, отчёты |
| `Документы_Сколково/` | Первоисточники и метаданные |
| `src/` | Код: `scraper`, `rag_service`, `mcp_server`, `admin`, `common` |
| `data/` | Векторная БД и индексы (не коммитится) |
| `deploy/` | Docker и конфиги развёртывания |

## Технологии

Go 1.25 · Qdrant · локальные эмбеддинги multilingual-e5 (TEI) · PostgreSQL/JSON · MCP (Streamable HTTP).

## Быстрый старт

```bash
cp .env.example .env
docker compose -f deploy/docker-compose.yml up -d   # Qdrant + TEI + Postgres
go build -o bin/skolkovo ./cmd/skolkovo
bin/skolkovo scrape    # скачать документы с dochub.sk.ru
bin/skolkovo index     # проиндексировать действующие
bin/skolkovo serve     # планировщик + MCP (:8080) + админка (:8090)
```

Полное руководство и подключение к MCP — [deploy/README.md](deploy/README.md).
