# База Сколково

RAG-база знаний по документам Фонда «Сколково» с **открытым MCP-доступом** для агентов и приложений, **системой управления резидентством** и **Telegram-ботом** для клиентов.

## Возможности

- 📥 **Сбор документов** с https://dochub.sk.ru/foundation/documents/
- 📅 **Мероприятия** — мониторинг и индексация мероприятий Сколково (RSS + HTML-скрапинг)
- 🏆 **Конкурсы и гранты** — парсинг конкурсов/грантов с HTML-страниц
- ❓ **FAQ** — сбор и индексация часто задаваемых вопросов
- 📱 **Telegram** — мониторинг Telegram-каналов Сколково через RSS-агрегаторы
- 🏢 **Управление резидентством** — клиенты, стадии, чек-листы, дедлайны, шаблоны документов
- 🤖 **Telegram-бот** — `/status`, `/deadlines`, `/docs`, `/ask` для клиентов резидентства
- 🧠 **RAG-база** — векторный поиск и извлечение по документам
- 🔌 **Открытый MCP-сервер** — 14 инструментов (документы, мероприятия, конкурсы, FAQ, резиденты, резидентство)
- 🛠 **Админка** — классификация, валидация, контроль статуса (действует / устарел)
- 📊 **Аналитика** — дашборд с Chart.js, экспорт в CSV
- 🔍 **Diff** — сравнение версий документов с LCS-алгоритмом
- ✅ **Validator** — проверка полноты базы с рекомендациями
- 🗄 **Migrate** — миграции схемы PostgreSQL

## Технологии

Go 1.26 · Qdrant · локальные эмбеддинги multilingual-e5 (TEI) · PostgreSQL · MCP (Streamable HTTP) · chromedp · Telegram Bot API · Chart.js

## Быстрый старт

```bash
cp .env.example .env
docker compose -f deploy/docker-compose.yml up -d   # Qdrant + TEI + Postgres
make build                                          # сборка единого бинаря
make scrape                                         # скачать документы с dochub.sk.ru
make index                                          # проиндексировать действующие
make serve                                          # планировщик + MCP (:8080) + админка (:8090)
```

### CLI команды

| Команда | Описание |
| :--- | :--- |
| `make scrape` | Запуск парсера документов |
| `make index` | Индексация действующих документов в RAG |
| `make sync` | Синхронизация всех источников (новости, мероприятия, конкурсы, FAQ, Telegram) |
| `make embed` | Переиндексация эмбеддингов |
| `make mcp` | Запуск MCP-сервера |
| `make admin` | Запуск админки |
| `make serve` | Полный режим: планировщик + MCP + админка |

Полное руководство и подключение к MCP — [deploy/README.md](deploy/README.md).
